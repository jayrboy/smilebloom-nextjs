## Home assets

วางไฟล์รูปสำหรับหน้า Home ไว้ในโฟลเดอร์นี้ โดยใช้ชื่อดังนี้ (คุณสามารถแทนที่ไฟล์เดิมได้เลย)

- `logo.svg` (หรือ `.png`): โลโก้ซ้ายบน/กลาง Hero
- `intro-1.(jpg|png|webp|svg)`: รูปประกอบใน section แนะนำ (รูปที่ 1)
- `intro-2.(jpg|png|webp|svg)`: รูปประกอบใน section แนะนำ (รูปที่ 2)
- `qr.(png|webp|svg)`: QR ติดต่อด้านล่าง

หมายเหตุ: ตอนนี้มีไฟล์ placeholder แบบ SVG ให้แล้ว เพื่อให้หน้าแสดงผลได้ทันที
